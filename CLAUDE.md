# EthicsNavi - 臨床倫理4分割AI相談

## 概要
Jonsenの臨床倫理4分割表（医学的適応/患者の意向/QOL/周囲の状況）に沿って、AIが質問を投げかけながら倫理的整理を支援。倫理カンファレンスの準備や、一人で悩む場面の思考整理ツール。

## ユーザープロフィール
- リハビリテーション科専門医
- 倫理観および周囲からの信頼を重視する医療者
- 株式会社Rehabit 経営者

## 技術スタック
- Python 3.11+
- Claude API (対話・分析エンジン)
- Streamlit (Web UI)

## Jonsenの4分割表
```
┌─────────────────────┬─────────────────────┐
│ 1. 医学的適応        │ 2. 患者の意向        │
│ (Medical Indications)│ (Patient Preferences)│
│ ・診断と予後         │ ・患者の希望         │
│ ・治療の目標         │ ・インフォームドコンセント │
│ ・治療の選択肢       │ ・判断能力           │
│ ・医学的無益性       │ ・代理決定者         │
├─────────────────────┼─────────────────────┤
│ 3. QOL              │ 4. 周囲の状況        │
│ (Quality of Life)    │ (Contextual Features)│
│ ・治療後のQOL見込み   │ ・家族の意向         │
│ ・日常生活への影響    │ ・経済的問題         │
│ ・QOL評価の主体      │ ・法的問題           │
│ ・偏見の排除         │ ・施設の方針         │
└─────────────────────┴─────────────────────┘
```

## 対話フロー
1. ケース概要の入力（自由記述）
2. AIが4象限それぞれについて深掘り質問
3. 整理された4分割表の出力
4. 論点の整理と検討すべきポイントの提示
5. PDF/印刷用レポート出力（倫理カンファレンス資料用）

## 重要な設計方針
- AIは「答え」を出さない。思考整理を支援する
- 判断はあくまで医療チームが行う
- 免責事項を明示（「本ツールは意思決定支援であり、最終判断は医療チームに委ねられます」）
